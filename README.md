# BedwarsOverlay

### How to use:
- To use the app you need a Hypixel API key to input in the setting from https://developer.hypixel.net/.
- When the app is running and you are in a Bedwars game simply run `/who` in game and it will display all the players in the app. 


### Info:
- You can use a hotkey (`J` by default) to hide and unhide the app, if the app is hiden when you run `/who` it will unhide automaticly.
- To have the [Minecraft font](https://www.dafont.com/minecraft.font) like in the image bellow you have to download it on your PC and selected it in the settings.
